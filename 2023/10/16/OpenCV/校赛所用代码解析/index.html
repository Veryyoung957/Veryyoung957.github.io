校赛视觉代码

  1.  cap.read//图像读取

  2.  cvtColor（InputArray src，outputArray dst，int code，int dstCn=0）//图像空间转换
![](https://img2020.cnblogs.com/blog/922321/202112/922321-20211211092943135-1639333672.png)

  3.    inRange()(InputArray src, InputArray lowerb,  
                              InputArray upperb, OutputArray dst);//范围内的值二值化

  • src:输入图像

  • dst:输出图像

  • ksize:高斯内核大小

  • borderType = BORDER_DEFAULT:像素外推法选择标志，取值范围在表于5 中给出， 默认参数为BORDER_DEFAULT，表示不包含边界值倒序填充.

  1.  adaptiveThreshold( InputArray src, OutputArray dst
                                  double maxValue, int adaptiveMethod,
                                  int thresholdType, int blockSize, double C );//自适应二值化`

  • src表示需要进行二值化的图像;需要注意的是,该输入必须是8-bit单通道的图像;

  • dst表示输出图像的二值图像;

  • maxValue是一个非零值,用于对哪些满足条件的阈值进行赋值;

  • adaptiveMethod表示选择哪一种自适应阈值算法;Opencv提供两种,ADAPTIVE_THRESH_MEAN_C与ADAPTIVE_THRESH_GAUSSIAN_C,下面会详细介绍;

  • thresholdType表示二值化类型,OpenCV提供两种, THRESH_BINARY与THRESH_BINARY_INV,;

  • blocksize表示参与计算的像素的领域范围,必须使用奇数;

  • C可以为正数, 零或者负数;用于在计算过程中容忍程度;

  1.  dilate( InputArray src, OutputArray dst, InputArray kernel,
                       Point anchor = Point(-1,-1), int iterations = 1,
                       int borderType = BORDER_CONSTANT,
                       const Scalar& borderValue = morphologyDefaultBorderValue() );//膨胀

  • src:输入图像；

  • dst:输出图像；

  • kernel:膨胀核元素;

  • anchor = Point(-1,-1):膨胀锚点位置，(-1,-1)是默认在中心位置；

  • borderType：像素外推法；

  • borderValue：固定边缘情况下的边缘值；

  1.  findContours( InputArray image, OutputArrayOfArrays contours,
                           OutputArray hierarchy, int mode,
                           int method, Point offset = Point());//寻找轮廓

  • image:输入的单通道图像，可以是灰度图像，更多的是二值化图像

  • contours:输出的轮廓，定义为vector<vector> contours，是一个双重向量数组

  • hierarchy:定义为vector<Vec4i> hierarchy ，是描述轮廓关系的向量；可以不返回这个值；

  • mode:int型的mode，定义轮廓的检索模式：

               取值一：CV_RETR_EXTERNAL**只检测最外围轮廓**，包含在外围轮廓内的内围轮廓被忽略

               取值二：CV_RETR_LIST   检测所有的轮廓，包括内围、外围轮廓，但是检测到的轮廓不建立等级关

                      系，彼此之间独立，没有等级关系，这就意味着这个检索模式下不存在父轮廓或内嵌轮廓，

                      所以hierarchy向量内所有元素的第3、第4个分量都会被置为-1，具体下文会讲到

               取值三：CV_RETR_CCOMP  检测所有的轮廓，但所有轮廓只建立两个等级关系，外围为顶层，若外围

                      内的内围轮廓还包含了其他的轮廓信息，则内围内的所有轮廓均归属于顶层

               取值四：CV_RETR_TREE， 检测所有轮廓，所有轮廓建立一个等级树结构。外层轮廓包含内层轮廓，内

                       层轮廓还可以继续包含内嵌轮廓。

  • method:int型的method，定义轮廓的近似方法：

               取值一：CV_CHAIN_APPROX_NONE 保存物体边界上所有连续的轮廓点到contours向量内

               取值二：CV_CHAIN_APPROX_SIMPLE 仅保存轮廓的拐点信息，把所有轮廓拐点处的点保存入contours

                       向量内，拐点与拐点之间直线段上的信息点不予保留

               取值三和四：CV_CHAIN_APPROX_TC89_L1，CV_CHAIN_APPROX_TC89_KCOS使用teh-Chinl chain 近似算法

  • Point:偏移量

  1.  drawContours( InputOutputArray image, InputArrayOfArrays contours，
                           int contourIdx, const Scalar& color,
                           int thickness = 1, int lineType = LINE_8,
                           InputArray hierarchy = noArray(),
                           int maxLevel = INT_MAX, Point offset = Point() );//绘制轮廓

  • image:需要绘制轮廓的图像

  • contours:需要绘制的轮廓

  • image：目标图像

  • contoursIdx：指定绘制轮廓的下标（若为负数，则绘制所有轮廓）

  • color：绘制轮廓的颜色

  • thickness：绘制轮廓的线的宽度（若为负数，则填充轮廓内部）

  • lineType：绘制轮廓的线型（4连通、8连通或者反锯齿）

  1. arcLength( InputArray curve, bool closed )//返回轮廓长度

  • curve:输入二维点集

  • closed:是否闭合

  1. approxPolyDP( InputArray curve,
                            OutputArray approxCurve,
                            double epsilon, bool closed )//多边形拟合

  • curve:输入二维点集可为Mat或Vector

  • approxCurve:多边形逼近结果

  • epsilon：逼近的精度，设定的原始曲线与近似曲线之间的最大距离，即上文提到的的阈值。

  • closed：如果为真，则近似的曲线为封闭曲线（第一个顶点和最后一个顶点相连），否则，近似的曲线不封闭。

  1. boundingRect( InputArray array )//输出最小包含矩形
  • array:输入的点集
  1. rectangle(InputOutputArray img, Rect rec,
                      const Scalar& color, int thickness = 1,
                      int lineType = LINE_8, int shift = 0)//绘制矩形

  • img:输入需要绘制的图像；

  • rec:要绘制矩形的4个点的坐标；

  • color:所绘制线的颜色；

  • tickness:线的宽度；

  • lineType:线的类型；

  • shift:点坐标中的小数位数;

  1. putText( InputOutputArray img, const String& text, Point org,
                     int fontFace, double fontScale, Scalar color,
                     int thickness = 1, int lineType = LINE_8,
                     bool bottomLeftOrigin = false )//写字函数

  • img:输入图像

  • text:需要输入的文字

  • org:文字左下角坐标;

  • ontFace:字体类型

  • fontScale:字体结构

  • color:字体颜色；

  • thickness:字体宽度;

  • lineType:线类型

  • bottomLeftOrigin:图像数据原点
